# zn

📝 Zettelkasten utility

## Usage

```
Usage: zn [--help] [--version] title

Positional arguments:
  title          zettel title [nargs=0..1] [default: ""]

Optional arguments:
  -h, --help     shows help message and exits
  -v, --version  prints version information and exits
```

## Requires
